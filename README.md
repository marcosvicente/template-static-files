# Template static file

The template is create fot static files.

  - Javascript minify
  - Css minify
  - Image minify
  - Font minify
  - Es6 Babel
  - Basic sass configuration
  - 
# Instalation
```sh
$ npm install --global gulp-cli@4.0.2
$ git clone ...
$ npm install 
```
### Usage
> Sass
Run the file sass
```sh
$ gulp sass
```
> Sass wacth
Run the watch file sass 
```sh
$ gulp watch
```
> Image min
Run for minify image
```sh
$ gulp imagemin
```
> Css minify
Run for minify all css files
```sh
$ gulp cssmin
```
> Html minify
Run for minify all html files
```sh
$ gulp htmlmin
```
> JS minify
Run for minify all js files
```sh
$ gulp jsmin
```
> Font minify
Run for minify all font files
```sh
$ gulp font
```
> Minify all files
Run for minify all files
```sh
$ gulp default
```
> Clean dist
 Delete all file of path dis
```sh
$ gulp clean
```
> Run babel es6
Run for babel es6
```sh
$ gulp compile-es6
```
